\# Quantum vs Classical Stock Prediction System



This project presents a comparative study between classical computational methods and quantum computing approaches for modeling stock behavior. It demonstrates how deterministic simulations differ from probabilistic quantum models using Qiskit.



\---



\## 🚀 Key Features



\- Classical simulation of stock price behavior using percentage-based modeling  

\- Quantum circuit implementation using Qiskit  

\- Visualization of quantum states and probability distributions  

\- Comparative analysis between classical and quantum approaches  

\- Demonstration of superposition, entanglement, and measurement  



\---



\## 🧠 Methodology



\### 🔹 Classical Model

\- Generated simulated stock percentage changes over time  

\- Applied rolling window analysis (e.g., 90-day intervals) to estimate future prices  

\- Implemented rule-based decision strategy for buying shares based on predicted price thresholds  

\- Evaluated portfolio performance based on budget constraints and stock behavior  



\### 🔹 Quantum Model

\- Constructed quantum circuits using Qiskit  

\- Applied \*\*Hadamard gates\*\* to create superposition  

\- Used \*\*CNOT gates\*\* to introduce entanglement between qubits  

\- Executed circuits using \*\*AerSimulator\*\*  

\- Collected measurement results and converted quantum states into percentage-based stock changes  

\- Visualized outcomes using histograms and circuit diagrams :contentReference\[oaicite:0]{index=0}

🔍 Key Insights
